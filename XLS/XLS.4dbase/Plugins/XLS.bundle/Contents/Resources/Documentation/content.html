XLS

Plugin Identifier: 20000

Summary

This 4D plugin allows the creation of XLS documents using the xlslib 2.3.4 library.

The spreadsheet is exported in BIFF8 (Unicode) format.

iconv is included in the bundle.

Workbook

XLS WORKBOOK CLEAR

XLS WORKBOOK CLEAR (book)

book (LONGINT)

Reference to the workbook.

Discussion

Clear the workbook object from memory.

XLS WORKBOOK Create

book:=XLS WORKBOOK Create

book (LONGINT)

Reference to the workbook.

Discussion

Create a new workbook. You should clear the reference with XLS WORKBOOK CLEAR when you are done.

XLS WORKBOOK Create area node

node:=XLS WORKBOOK Create area node (book; cell1; cell2; address; mode)

book (LONGINT)

Reference to the workbook.

cell1 (LONGINT)

Reference to a cell.

cell2 (LONGINT)

Reference to another cell.

address (LONGINT)

You can use a XLS Cell Reference constant for address.

mode (LONGINT)

You can use a XLS Cell Interpretation constant for mode.

node (LONGINT)

Reference to the node.

Discussion

Create an node specified by two cells. You should release the reference with XLS CELL RELEASE when you no longer need to reference the expression; otherwise, the number of references will keep increasing and take longer for the plugin to resolve.

XLS WORKBOOK Save document

error:=XLS WORKBOOK Save document (book; path)

book (LONGINT)

Reference to the workbook.

path (TEXT)

Full document path with extension.

error (LONGINT)

  • FILE_ERROR (-1)
  • GENERAL_ERROR (-100)

Discussion

Export the workbook document to path.
